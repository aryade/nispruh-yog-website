# Images Directory

This folder contains all images for the Nispruh Yog website.

## Folder Organization

### `/teacher`
Store teacher/instructor profile photos here
- `profile-photo.jpg` - Main profile picture
- `teaching.jpg` - Teaching/instructing photo
- `headshot.jpg` - Professional headshot

### `/practices`
Store practice and technique photos here
- `meditation.jpg`
- `breathing-technique.jpg`
- `chanting.jpg`
- `visualization.jpg`

### `/events`
Store event and retreat photos here
- `retreat-2024.jpg`
- `workshop-location.jpg`
- `group-photo.jpg`
- `meditation-circle.jpg`

### `/blog`
Store blog article and teaching photos here
- `meditation-basics.jpg`
- `yoga-benefits.jpg`
- `breathing-techniques.jpg`
- `practice-guide.jpg`

### `/locations`
Store location and studio photos here
- `studio.jpg` - Studio exterior/interior
- `meditation-room.jpg` - Meditation space
- `outdoor-class.jpg` - Outdoor practice space
- `map-location.jpg` - Location photo for map

## File Naming Guidelines

✅ Good names:
- `teacher-profile.jpg`
- `meditation-class.jpg`
- `retreat-location.jpg`

❌ Bad names:
- `photo.jpg`
- `image1.jpg`
- `teacher profile.jpg` (spaces)

## Image Requirements

- **Format:** JPG, WebP, or PNG
- **Max Size:** 1MB (compress before uploading)
- **Recommended Dimensions:** See EDITING_GUIDE.md for specific sizes
- **Quality:** High resolution (1200px+ width for web use)

## How to Add Images

1. Place your image file in the appropriate folder
2. Use the correct path in components:
   ```
   /images/teacher/profile.jpg
   /images/practices/meditation.jpg
   /images/events/retreat.jpg
   /images/blog/article-name.jpg
   /images/locations/studio.jpg
   ```

3. Always add alt text describing the image

## Compression Tools

Before uploading large images, compress them:
- https://tinypng.com
- https://squoosh.app
- https://imagecompressor.com
