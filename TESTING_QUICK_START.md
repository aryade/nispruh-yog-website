## 🚨 Issues You Might Find

### Common Issues

**"Page takes 5 seconds to load"**
- Check images are optimized
- Check network requests in DevTools
- Consider performance optimization

**"Dark mode text is hard to read"**
- Check color contrast (should be 4.5:1)
- Adjust text color in CSS
- Use contrast checker tool

**"Form doesn't submit on mobile"**
- Check button size (should be 44x44px minimum)
- Check form styling on mobile viewport
- Test on actual phone

**"Can't navigate with keyboard"**
- Check tabindex attributes
- Ensure all interactive elements focusable
- Add focus styling

---

## 💡 Pro Tips

1. **Test on real devices** - Emulation isn't perfect
2. **Clear cache** - Prevents testing old versions
3. **Use incognito mode** - Clean browser state
4. **DevTools is your friend** - Use Network, Performance, Elements tabs
5. **Document issues** - Screenshot problems with comments
6. **Test one thing at a time** - Don't test multiple things simultaneously
7. **Read error messages** - They often tell you what's wrong
8. **Test edge cases** - Long text, special characters, slow networks
