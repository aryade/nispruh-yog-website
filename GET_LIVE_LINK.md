## 🚀 Deploy to Vercel (10 Minutes)

### **Step 1: Go to Vercel**

1. Open: https://vercel.com
2. Click **"Sign Up"** button (top right)

### **Step 2: Sign Up with GitHub**

1. Click **"Continue with GitHub"**
2. Enter your GitHub username: `aryade`
3. Enter your GitHub password
4. Click **"Authorize vercel"**

### **Step 3: Create New Project**

1. Click **"New Project"** button
2. Click **"Select a Git Repository"**

### **Step 4: Select Your Repository**

1. Search for: `nispruh-yog-website`
2. Click on it to select it
3. Click **"Import"**

### **Step 5: Configure Project**

Leave all settings as default:
- Framework Preset: `Next.js`
- Build Command: `next build`
- Output Directory: `.next`
- Install Command: `npm install`

Click **"Deploy"** button

### **Step 6: Wait for Deployment**

Vercel will:
1. Build your website (2-3 minutes)
2. Run tests
3. Deploy it live

You'll see a progress bar and status updates.

### **Step 7: Get Your Live URL! 🎉**

When deployment is complete, you'll see:

```
✅ Deployment successful!
Visit your live site: https://nispruh-yog-website.vercel.app
```

**That's your live website URL!**

## 🎯 Optional: Add Custom Domain

Your default link is: `https://nispruh-yog-website.vercel.app`

If you want a custom domain like `nispruhyog.com`:

### **Step 1: Buy Domain**

1. Go to: https://godaddy.com or https://namecheap.com
2. Search for your domain (e.g., `nispruhyog.com`)
3. Add to cart and checkout
4. Cost: ~$15/year
5. Wait for confirmation email

### **Step 2: Connect to Vercel**

1. In Vercel dashboard, go to **Settings** → **Domains**
2. Enter your domain name: `nispruhyog.com`
3. Vercel gives you DNS settings
4. Go to your domain registrar (GoDaddy/Namecheap)
5. Update DNS settings with Vercel's values
6. Wait 24-48 hours for changes to propagate

**Your website will then be at:** `https://nispruhyog.com`

## 🆘 Troubleshooting

### ❌ "Build Failed" on Vercel

**Problem:** Error during deployment
**Solution:**
1. Check the error message in Vercel logs
2. Fix the error locally
3. Test with `npm run dev`
4. Push to GitHub
5. Vercel automatically redeploys

### ❌ "Domain Not Available"

**Problem:** Can't add custom domain
**Solution:**
1. Wait 24-48 hours for DNS to propagate
2. Verify DNS settings are correct
3. Try clearing browser cache
4. Try incognito mode

### ❌ "Site Not Loading"

**Problem:** URL doesn't work
**Solution:**
1. Check deployment status in Vercel
2. Wait for deployment to complete
3. Try refreshing page (Ctrl+R)
4. Check internet connection
